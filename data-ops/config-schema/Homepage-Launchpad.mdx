---
title: "Homepage Launchpad"
description: "The Homepage Launchpad is shown on the homepage of your project for all of the
project members. You can pin important resources and add a description using
markdown formatting."
---

## Schema
Data models define the following properties:


<ParamField path="glean" type="1.0">
    The Glean file format version.
</ParamField>
<ParamField path="grn" type="string">
    The persistent identifier for this homepage launchpad.
</ParamField>
<ParamField path="type" type="homepage_launchpage">
    The type of this resource, which in this case should always be "homepage_launchpad".
</ParamField>
<ParamField path="description" type="string">
    The description to display on the homepage launchpad.
</ParamField>
<ParamField path="pinnedResourceIds" type="array">
    The resources to pin on the homepage launchpad. This should be an array of GRNs and/or file paths to resources.
</ParamField>

## Example
```yaml
glean: "1.0"
type: homepage_launchpad
description: |
  ## Cool project

  A good starting point is our KPIs:
  - one
  - two
  - three
pinnedResourceIds:
  - ./orders.yml
  - ./domesticOrdersChart.yml
  - dsb:z3rnvYe6YwZcUSZh
  - sv:Jv8gpgYORO6VSdWe
```
