## Schema

Color palettes define the following properties:

<ParamField path="glean" type="1.0">
    The Glean file format version.
</ParamField>
<ParamField path="type" type="color_palette">
    The type of this resource, which in this case should always be "color_palette".
</ParamField>
<ParamField path="name" type="string">
    The user-facing name for this color palette
</ParamField>
<ParamField path="grn" type="string">
    The persistent identifier for this color palette.
</ParamField>

<ParamField path="defaultColor" type="CustomColor">
    The default custom color of this color palette.
</ParamField>
<ParamField path="Color" type="<array>CustomColor">
    The custom colors of this color palette.
</ParamField>
<ParamField path="options" type="object">
</ParamField>

<Expandable title="properties">

<ParamField path="additionalColorGeneration" type="interpolate| repeat">
    Defines how additional color values are generated after the provided palette of colors is exhausted.
</ParamField>

</Expandable>

## CustomColor
The custom colors of this color palette.

<ParamField path="name" type="string">
    The user-visible name of this color value.
</ParamField>
<ParamField path="color" type="string">
    A CSS color value, generally a HEX value.
</ParamField>

## Example

```yaml
glean: "1.0"
name: Twilight
type: color_palette
defaultColor:
  color: "#234990"
colors:
  - color: "#003F5C"
  - color: "#444E86"
  - color: "#955196"
  - color: "#DD5182"
  - color: "#FF6E54"
  - color: "#FFA600"
options:
  additionalColorGeneration: interpolate
```
